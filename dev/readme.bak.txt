== Description ==

This plugin has been created to display beautiful and functional grid galleries with minimum configuration effort and without slowing down your page load.

Go Gallery is a powerful responsive grid gallery wordpress plugin. You can change spacings, frames, colors, etc. by simply adjusting settings directly in shortcodes. Unlimited number of galleries with different settings can be used on any page or in posts.

Go Gallery is optimized for speed. Configuration is fast and simple, everything can be done just by putting gallery shortcode(s) to the page.

Go Gallery enables categories in media images and adds filter menu to each gallery for easy and beautiful sorting. 

= Features: =

 * Responsive, optimized for mobile
 * Editable pop-up responsive grid template
 * Full site lock with signup / login pop-up
 * Triggered by click or by time delay
 * Whitelist, keep specified pages locked/unlocked

== Installation ==

= Minimum Requirements =

 * WordPress 3.8+
 * PHP version 5.2.4+
 * MySQL version 5.0+

**This section describes how to install the plugin and get it working**


= Automatic installation (easiest way) =

To do an automatic install of ReForce PopUp, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "ReForce PopUp" and click Search Plugins. Once you have found it you can install it by simply clicking "Install Now".

= Manual installation =

**Uploading in WordPress Dashboard**

1. Download `reforce-lite.zip`
2. Navigate to the 'Add New' in the plugins dashboard
3. Navigate to the 'Upload' area
4. Select `reforce-lite.zip` from your computer
5. Click 'Install Now'
6. Activate the plugin in the Plugin dashboard

**Using FTP**

1. Download `reforce-lite.zip`
2. Extract the `reforce-lite` directory to your computer
3. Upload the `reforce-lite` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard

The WordPress codex contains [instructions on how to install a WordPress plugin](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
